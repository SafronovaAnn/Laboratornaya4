# Лабораторная работа №4
## Последовательные одномерные контейнеры

**Язык программирования:** C++, Rust, Go

---

## Описание лабораторной работы

Цель работы: изучение алгоритмов формирования и обработки одномерных массивов и последовательных контейнеров, программирование и отладка программ формирования и обработки массивов.

| `task1.cpp` | Задание №1 | Решение нелинейного уравнения x² - 3·sin(x) = 0 методами половинного деления, Ньютона и простых итераций |
| `task2.cpp` | Задание №2 | Работа с массивами (вариант 2): поиск элементов, циклический сдвиг, простые числа |
